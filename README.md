# Projet Système Réparti — Scaffold

Scaffold minimal pour débuter le projet pédagogique : frontend, backend, base PostgreSQL, manifestes Kubernetes, playbook Ansible et Jenkinsfile.

Pré-requis : Docker, Docker Compose.

Démarrer en local (Docker Compose):

```powershell
cd C:\ProjetSystemRepartie
docker-compose up --build
```

Le frontend sera accessible sur http://localhost:8080 et l'API sur http://localhost:5000.
